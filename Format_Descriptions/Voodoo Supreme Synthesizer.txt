Footer:

Search for the VSS0 mark between the last 64 bytes in the file

   0      4      Mark (VSS0)
   4      4      Number of subsongs (NSS)
   8  NSS*4      Subsong offset table. Add this number to the offset of the
                 mark to find the offset to the subsong structure (and start
                 of other structures)


Sample data:

A big block with all the sample data. First comes 2 bytes with the length of
the samples, then the sample data itself.


Track data:

When an argument is an offset, you take the argument and multiply with 4 and
add this to the start offset of the offset table for the current subsong.

00-7F xx    Note. xx is how many ticks to wait until reading next track
            command
81 xx       Gosub to track. Plays the track at offset xx from offset table
82          Return. Continue right after the last gosub
83 xx yy    Start loop. xx is the loop count. yy not used
84          Loop. Goto right after the last "start loop" if count has not
            reached 0
85 xx yy    Set sample 1 at offset xx and sample 2 at offset yy
86 xx       Use volume envelope table at offset xx
87 xx       Use period command table at offset xx
88 xx yy    Use waveform at offset xx and mode yy
89 xx yy zz Portamento. xx = start note. yy = stop note. zz = number of
            ticks to do the slide
8A xx       Set note transpose to xx
8B xx       Go to track at offset xx. End of track
8C xx       Set reset flags to xx
8D xx       Set waveform mask to xx
FF xx       Note cut. Stops the current playing note. xx is how many ticks
            to wait until reading next track command


Offsets:

A list of signed offsets to either tracks or others structures / tables.


Here is a description of the different structures.


Subsong structure:

   0      4      Offset to voice 0 track data
   4      4      Offset to voice 1 track data
   8      4      Offset to voice 2 track data
   C      4      Offset to voice 3 track data


Volume envelope table:

The envelope table contains different values or special markers. The markers
are:

88 xx Jump to position xx

If not a marker, the point data is as the following:

xx yy If yy = 0:
        xx = volume level
      If yy <> 0:
	    xx = Delta value
		yy = Speed


Period command table:

This table contains different commands:

7E xx ?
7F xx Set the current period one octave up. Set tick counter to xx
FE xx Set the current period one octave down. Set tick counter to xx
FF xx Jump to position xx


Waveform command table:

The content of the table depends on the mode set.

2C  4 Pointer to voice 2 old interrupt handler

38  4 Pointer to voice 1 synthesis audio buffer (2 * 32 bytes)
3C  4 Pointer to voice 2 synthesis audio buffer (2 * 32 bytes)
40  4 Pointer to voice 3 synthesis audio buffer (2 * 32 bytes)

48  1 Flags
      0 = Channels allocated
	  1 = Audio buffer allocated
	  2 = Interrupt handlers has been set
